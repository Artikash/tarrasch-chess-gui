Instructions for building portable .zip distribution

1) Copy the release executable from .\vc_msw\Tarrasch.exe to this directory.
2) Make sure file bin.zip doesn't exist in this directory.
3) Select Tarrasch.exe, Tarrasch.ini (empty file), book.pgn, basic.pgn, and the Engines and web subdirectories
Right click and choose 7zip "Add to bin.zip".
4) Rename bin.zip appropriately, eg to tarrasch-bin-V3 alpha 2015-12-11-a.zip.
